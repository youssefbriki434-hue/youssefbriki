## 🚀 Featured Project

### 📦 Internal Material Request Management System
> Final-year project built during an internship at **Poulina Group Holding**, one of Tunisia's largest industrial conglomerates.

A full-stack web application that digitizes and streamlines the internal material request workflow across departments.

**Key features:**
- Role-based access control (employees, managers, admins)
- Request creation, tracking, and approval workflow
- Email notifications via SMTP
- Dashboard with request status overview
- Built with **Django**, **MySQL**, **HTML/CSS** — no external UI frameworks

**What I learned:** Designing systems used by real employees under real business constraints. Managing a full development cycle from requirements gathering to deployment.
